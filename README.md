# General Assembly JavaScript (non-interactive) WAR card game

## Getting Started

1. Clone this git app:
```sh
git clone https://git.generalassemb.ly/doperry/wargame/war.js
```
2. Install nodejs
3. Run the code 

### Prerequisites

* [nodejs](https://nodejs.org/en/)
